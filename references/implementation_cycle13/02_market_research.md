# Market Research - Cycle 13 (2026-02-26)

## Focus
Travel planning UX patterns that improve clarity, trust, and decision flow.

## Sources reviewed
- Booking.com web flow patterns (hero + search-first + trust copy): https://www.booking.com/
- Airbnb booking clarity patterns (stepwise decisions and pricing visibility): https://www.airbnb.com/
- Stripe dashboard status/badge readability patterns for event feeds: https://stripe.com/

## Patterns extracted
1. Keep core planning actions prominent and grouped by user intent.
2. Show operational status with high-contrast badges at top of each event item.
3. Default to compact recent activity and provide expansion for history.
4. Keep pricing visible and always synchronized with the latest active selection.
5. In multi-step confirmations, provide explicit undo/reset path.
